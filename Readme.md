# 🌦️ Weather App

A responsive Weather App built with React.js that provides real-time weather information using the OpenWeatherMap API.

## 🚀 Features

- Search weather by city name
- Real-time temperature
- Humidity
- Wind speed
- Dynamic weather icons
- Responsive UI

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- Vite
- OpenWeatherMap API

## ⚙️ Installation

```bash
git clone https://github.com/pranavsharma245/weather-app.git

cd weather-app

npm install

npm run dev
```

Create a `.env` file:

```env
VITE_API_ID=YOUR_OPENWEATHER_API_KEY
```
